Description

The ImportDir and ImportThunk classes provide functionality for working with the import directory of a PE file.

See the PE and COFF Specification for more information on working with the PE file format.

The ImportDirList class provides STL-compatible iterators for import directory enumeration. The ImportThunkList class provides STL-compatible iterators for import thunk enumeration.
